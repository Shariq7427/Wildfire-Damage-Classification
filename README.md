# 🔥 Structure Wildfire Damage Classification

This project uses the **Structure Wildfire Damage Classification Dataset**, which contains aerial and ground-level images of buildings affected by wildfires in California (2020–2022). The goal is to classify structural damage into severity levels using machine learning / deep learning models.

---

## 📊 Dataset Overview

- Source: Hugging Face Dataset (CAL FIRE-based)
- Size: ~18,700 images
- Classes: 6 damage categories
- Format: Image + label classification

### 📌 Class Labels

| Label ID | Class Name     | Description |
|----------|---------------|-------------|
| 0        | affected      | 1–9% damage |
| 1        | destroyed     | >50% damage / fully destroyed |
| 2        | inaccessible  | cannot be assessed |
| 3        | major         | 26–50% damage |
| 4        | minor         | 10–25% damage |
| 5        | no_damage     | no visible damage |
